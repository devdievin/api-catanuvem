Api Catanuvem

Api para previsão climática. Busque pelas coordenadas ou pelo nome da cidade

Exemplo:

Coordenadas: https://api-catanuvem.vercel.app/weather/today/loc/-15.7801&-47.9292
Nome da cidade: https://api-catanuvem.vercel.app/weather/today/city/Brasília/DF
